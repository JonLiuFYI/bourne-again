# bourne-again 0.0
A broken robot regains its abilities. A game made in [Pyxel](https://github.com/kitao/pyxel).

Made for Global Game Jam 2020 at Shopify, in Ottawa.

## Requirements
* Pyxel 1.2.10
* Python 3.8
* Something to edit Python code in
